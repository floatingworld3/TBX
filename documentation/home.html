T B X V . 2

TBX is a utility for converting tracking data (usually facial performance capture data) into FBX files for use in 3d animation packages such as Maya, 3ds Max, Blender, Unreal,Unity and others. It is primarily designed to work in conjunction with the online tracking service at www.RigVadar.com.


Availble for Windows & Mac

ADVANTAGES OF THE RIG VADAR SYSTEM OVER BESPOKE BLEND SHAPE SOLUTIONS

Using the ARKit with an iPhone leads to latency issues, and it's recommended technique for using supplied video material is cumbersome and further increases delays.

There are also privacy as well as copyright issues that users can run into, as iPhone saves data to the cloud whilst covertly recording biometric data that users tacitly authorize.

Furthermore the proprietary software that utilizes the facial expressions output of the ARKit, requires complex and time consuming calibration and adjustment, especially around lip movements such as lip curl.

But the biggest drawback is that it can not animate a rig controlled by bones skinned to the mesh. It can not animate a mesh that has no morphs, which is a disadvantage as this type of rig is the easiest to set up and animate quickly.

The solution is a simple webcam or video based approach which records actual physical movement rather than detecting particular micro expressions triggered by the ARKit algorithm.

The rigs created by the Rigvadar app can be up and running in minutes, and can be refined into morphed meshes easily.

The FBX markers are deployed to control spline controls which in turn modulate morph targets as well as bones.

This preferred workflow utilizes TBX,which, in conjunction with the online performance capture service at RigVadar.com, produces a cross platform solution in the form of an FBX file - which is portable across all Autodesk products and elsewhere, quickly producing natural looking results faithful to the orginal, capturing facial nuances and head movements without the need for excessive editing or calibration.

FEATURES OF TBX

Converts the tracking data produced by the online tracker(including any additional points) into an FBX file for export to many 3d apps.

Audio file can be added to the FBX file or extracted from source video.

Output file location user-definable

The FBX animation can be clipped at any point, leaving static frames either side of the chosen frames, in order to create layered animations from different takes or sections.

Markers in the FBX can be resized and the UP vector chosen.

The audio and folder of separated images can be extracted from the source video.

Mask (outline of face) position can be tracked or if required, the root can be static.
